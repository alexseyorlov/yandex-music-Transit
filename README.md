# yandex-music-Transit (v5.0 API-Only) 🚀

Быстрый экспорт плейлистов Яндекс Музыки в TXT без скроллинга и без привязки к верстке страницы.

## 📋 Описание
Текущая версия `v5.0` использует API-first подход: скрипт перехватывает и обрабатывает JSON-ответы playlist API прямо в браузере, используя вашу текущую авторизованную сессию. Это дает высокую скорость и устойчивость по сравнению с DOM-сканированием виртуального списка.

## ✨ Особенности
- **API-only архитектура:** без двухфазного скролла и проблем Virtuoso.
- **Быстрый сбор:** обычно за секунды после открытия плейлиста.

- **Устойчивый парсинг:** поддержка разных форматов URL (`/users/.../playlists/...` и `/playlists/...`).
- **Дедупликация:** удаление дублей по ID и строке трека.
- **Удобная панель:** статусы `INIT`, `PROBE`, `CAPTURE`, `PARTIAL`, `COMPLETE`, `EMPTY`.
- **Контроль в один клик:** кнопки `Save TXT` и `Stop`.

## 🚀 Как использовать
1. Откройте нужный плейлист в веб-версии Яндекс Музыки (например, "Мне нравится").
2. Откройте инструменты разработчика: `F12` или `Ctrl+Shift+I`, вкладка **Console**.
3. Скопируйте код из [script.js](./script.js), вставьте в консоль и нажмите `Enter`.
4. Дождитесь статуса `CAPTURE`/`COMPLETE` или `PARTIAL`.
5. Нажмите **Save TXT** для скачивания списка.

## 🧭 Подсказки
- Если отображается `EMPTY`, обновите страницу плейлиста и запустите скрипт снова.
- Если видите `PARTIAL`, можно сохранить промежуточный результат через `Save TXT`.
- Кнопка `Stop` возвращает оригинальный `fetch` в странице.
