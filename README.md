# yandex-music-Transit (v4.4 Gravity Drift) 🚀

Легкий и эффективный инструмент для экспорта ваших плейлистов из Яндекс Музыки в чистый текстовый список. 

## 📋 Описание
Этот скрипт решает проблему "ленивой загрузки" (Virtuoso) в Яндекс Музыке. Когда в плейлисте 3000+ треков, обычные методы захвата данных часто пропускают композиции. **yandex-music-Transit** использует двухфазный алгоритм для достижения 100% точности на максимальной скорости.

## ✨ Особенности
- **Двухфазное сканирование:** Быстрый спуск вниз и тщательный "реверсивный" подъем.
- **Чистый вывод:** Формат `Артист - Название`, без мусора, дублей и лишней информации.
- **Acid Minimal UI:** Встроенная панель управления с индикацией фаз и прогресс-баром.
- **Безопасность:** Работает только в вашем браузере, не требует паролей или токенов.

## 🚀 Как использовать
1. Откройте нужный плейлист в браузере (например, "Мне нравится").
2. Откройте инструменты разработчика: `F12` или `Ctrl+Shift+I` и перейдите во вкладку **Console**.
3. Скопируйте код из файла [script.js](./script.js) этого репозитория.
4. Вставьте код в консоль и нажмите `Enter`.
5. Нажмите **START** в появившейся панели и дождитесь завершения. (**ВАЖНО!** НЕ Закрывать и не переходить со страницы, пока скрипт не закончит работу)
